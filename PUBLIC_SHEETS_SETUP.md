# Google Sheets 공개 설정 가이드

## 현재 방식: CSV Export (인증 불필요)

대시보드는 이제 **Google Sheets API 인증 없이** CSV export URL을 통해 데이터를 가져옵니다.

## Google Sheets 공개 설정 (필수)

### 1단계: Google Sheets 문서 열기

```
https://docs.google.com/spreadsheets/d/1_laE9Yxj-tajY23k36z3Bg2A_Mds8_V2A81DHnrUO68/edit
```

### 2단계: 공유 설정

1. 우측 상단 **"공유"** 버튼 클릭
2. **"링크가 있는 모든 사용자"** 또는 **"공개"** 선택
3. 권한: **"보기 가능(뷰어)"** 선택
4. **"완료"** 클릭

### 3단계: 확인

공개 설정이 완료되면:
- 대시보드가 자동으로 데이터를 가져올 수 있습니다
- 인증 없이 작동합니다
- 시트가 업데이트되면 대시보드도 자동으로 최신 데이터를 표시합니다

---

## 장점

✅ **인증 불필요**: 서비스 계정, Private Key 설정 불필요  
✅ **간단한 설정**: Google Sheets 공개 설정만 하면 됨  
✅ **자동 업데이트**: 시트가 변경되면 대시보드도 자동 반영  
✅ **빠른 속도**: CSV 방식으로 빠르게 데이터 로드

- 민감한 정보가 있다면 공개 설정 전에 확인하세요
